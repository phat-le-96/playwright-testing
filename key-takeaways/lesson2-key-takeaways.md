## Java Script

- Dùng lệnh node để chạy 1 chương trình trong javascript
- Comment: Dùng // để comment về 1 function hay chức năng mà IDE sẽ không chạy đoạn đó
- Comment nhiều dòng: /* */
- Biến: biến có thể thay đổi
- Hằng: không thể thay đổi, dùng khi không có nhu cầu thay đổi hay dùng 1 lần
- var và let: var thì cho khai báo lại, let khi khai báo lại sẽ báo lỗi
- var có phạm vi global, let thì phạm vi theo từng nhóm
- Dùng const: giúp code an toàn hơn, dễ đọc
- Dùng let: khi chắc chắn giá trị đó cần gán lại
- Các kiểu dữ liệu: number, string, boolean,...
- Các kiểu tham chiếu: Objject, array
- Các toán tử so sánh: >, <, >=, <=, ===, !==
- Các toán tử logic: ||, ??, &&
- Toán tử một ngôi: x++, ++x
- Toán tử toán học: +, -, *, /
- Câu điều kiện: if, if...else, switch... case
- Vòng lặp: for, forEach, while, do...while
- Vòng lặp for: (<khơi tạo>, <điều kiện>, <cập nhật>)
